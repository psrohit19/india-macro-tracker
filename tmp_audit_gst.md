Gaps / flags:
- No gaps: all 83 target months (2017-07..2024-05) sourced from official releases actually fetched.
- Convention: dates = month the collection was reported FOR (matches existing file: 2024-06-01 = 1.738 = "June 2024 ₹1.74 lakh cr"). Jul-2017..Feb-2018 releases report "for month X (received in X+1 up to ~25th)" — assigned to month X as printed.
- Mar-2018 has no standalone monthly release (convention changed to receipt-month from Apr-2018). Used ₹89,264 cr from the official "Final Monthly collection figures of GST" release, PRID 1527288 (2 Apr 2018, March row) — same figure media reported as "March 2018 GST collection".
- Apr-2020 and May-2020 had no standalone PIB releases (COVID; return deadlines relaxed). Values ₹32,294 cr and ₹62,009 cr are as printed in the official Jun-2020 release (PRID 1635572). Later official tables restate them as ₹32,172 / ₹62,151 cr (PRID 1776181) — as-printed values kept per series basis.
- Known as-printed vs later-restated diffs (kept as-printed, by design): Jan-2021 1,19,847 (till 6PM 31.01.2021; final 1,19,875); Apr-2021 1,41,384 (later 1,39,708); May-2021 1,02,709 (collections till 4 Jun 2021 due to extended due dates; later restated 97,821). Early FY2017-18 months were also restated in later tables (e.g. PRID 1527288); series keeps each month's own print.
- Jan-2020 release internal inconsistency: headline ₹1,10,828 cr vs its own components summing 1,10,818; headline kept.
- Mar/Apr/May-2024 releases print headline only to 2 decimals (₹1.78/2.10/1.73 lakh cr); exact gross taken as sum of the same release's printed components: 1,78,484 / 2,10,267 / 1,72,739 cr.
- 2018-12 note: Dec-2018 print is ₹94,726 cr (not 94,725).
